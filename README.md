This is the main analysis file used in the manuscript Harvey et al, "Mapping the Functional Landscape of KCNQ1 to Define Ion Channel Mechanisms and Arrhythmia Risk".
The R file contains the main analysis code and the rest of the files are inputs into our outputs from the main analysis.

The file KCNQ1Scores2_9.11.25.csv is a dense master summary file of all data in the paper.
